# Anniversary Gift Website (Static)

## How to use
1. Unzip the folder
2. Open `index.html` in a browser (double click)

## Pages
- index.html (Home)
- gallery.html (Photo album / lightbox)
- story.html (Timeline)
- letter.html (Love note)

## Photos
All photos are in: `assets/photos/`
You can replace them with your own images (keep file names or update `assets/js/gallery_data.js`).

## Personalize
Edit:
- index.html (title)
- letter.html (Your Name, message)
- story.html (timeline items)
